# Rapid Consultancy Website

Static website for the Rapid Consultancy gold loan management UI.

## Structure

- `index.html` — main website page
- `css/styles.css` — styling for landing pages and app views
- `js/main.js` — view switching, calculator logic, modals, toast, and interactions

## Usage

Open `index.html` in a browser to preview the site.

For local development, use any static server such as `Live Server` in VS Code or run:

```bash
python3 -m http.server
```

Then open `http://localhost:8000`.
